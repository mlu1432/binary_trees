# Binary Trees project
This repository contains various binary tree operations implemented in C.
## Files
In this repository we have the following files:
### Core Files
1. `binary_tree_node.c`: Contains the implementation for creating a binary tree node.
2. `1-binary_tree_insert_left.c`: Contains the implementation for inserting a node as the left-child of another node.
3. `2-binary_tree_insert_right.c`: Contains the implementation for inserting a node as the right-child of another node.
4. `3-binary_tree_delete.c`: Contains the implementation for deleting an entire binary tree.
5. `4-binary_tree_is_leaf.c`: Contains the implementation for checking if a node is a leaf or not.
6. `5-binary_tree_is_root.c`: Contains the implementation for checking if a particular node is a root node.
7. `6-binary_tree_preorder.c`: Contains the implementation for traversing a binary tree in preorder.
8. `7-binary_tree_inorder.c`: Contains the implementation for traversing a binary tree in inorder.
9. `8-binary_tree_postorder.c`: Contains the implementation for traversing a binary tree in postorder.
10. `9-binary_tree_height.c`: Contains the implementation for calculating the height of a binary tree.
11. `10-binary_tree_depth.c`: Contains the implementation for calculating the depth of a binary tree.
12. `11-binary_tree_size.c`: Contains the implementation for calculating the size of a binary tree.
13. `12-binary_tree_leaves.c`: Contains the implementation for counting the leaves in a binary tree.
14. `13-binary_tree_nodes.c`: Contains the implementation for counting non-leaf nodes in a binary tree.
15. `14-binary_tree_balance.c`: Contains the implementation for measuring the balance factor of a binary tree.
16. `15-binary_tree_is_full.c`: Contains the implementation for checking if a binary tree is full.
17. `16-binary_tree_is_perfect.c`: Contains the implementation for checking if a binary tree is perfect.
### Auxiliary files
1. `17-binary_tree_sibling.c`: Contains a function that finds the sibling of a node.
2. `18-binary_tree_uncle.c`: Contains a function that finds the uncle of a node.
### Test Files
1. `tests`: Contains various test files for validating the correctness of individual operations.
